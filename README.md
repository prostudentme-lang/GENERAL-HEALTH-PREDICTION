# GENERAL-HEALTH-PREDICTION
Title

General Health Prediction Using Machine Learning

Introduction

Heart disease is one of the leading causes of mortality worldwide. Early prediction of cardiovascular disease using data-driven techniques can significantly reduce health risks. This project uses machine learning algorithms to predict general health status, particularly heart disease, based on lifestyle and medical attributes.

Objective

To analyze general health data from CVD_cleaned.csv

To build prediction models using Logistic Regression, SVM, and Random Forest

To compare model performance using evaluation metrics

To predict health status for a single individual input

Dataset Description

The dataset CVD_cleaned.csv contains health-related attributes such as age category, BMI, smoking history, alcohol consumption, exercise habits, and heart disease status.

Methodology

Data preprocessing and label encoding

Splitting data into 80% training and 20% testing

Training Logistic Regression, SVM, and Random Forest models

Evaluating models using accuracy score, confusion matrix, and classification report

Hyperparameter tuning using Randomized Grid Search

Predicting health status for a single input (AI simulation)

Tools & Technologies

Python

Google Colab

Pandas, NumPy

Scikit-learn

Matplotlib

Conclusion

The project successfully demonstrates that machine learning techniques can effectively predict heart disease. Among the models used, Random Forest achieved the highest accuracy.

Future Scope

Use of deep learning models

Integration with real-time health data

Development of healthcare mobile applications
